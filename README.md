# Short-Term Rental Analysis: Jersey City vs. Newark

## Overview
This project analyzes short-term rental pricing data from Airbnb listings in Jersey City and Newark using Microsoft Excel. The goal of the project was to compare pricing trends, availability, and listing distribution between both cities.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Data Visualization
- Data Cleaning Techniques

## Project Tasks
- Cleaned and organized raw Airbnb listing data
- Compared rental pricing between Jersey City and Newark
- Analyzed listing availability and market distribution
- Used Pivot Tables and Excel functions to summarize trends
- Created charts and visualizations to highlight differences across both markets

## Key Findings
- Jersey City rental listings were generally more expensive than Newark listings.
- Entire homes/apartments had significantly higher prices than private or shared rooms.
- Rental prices tended to increase with the number of rooms, although some inconsistencies appeared among larger listings.
- In Jersey City, Ward E had the highest average rental prices, while Ward A had the lowest.
- In Newark, the East Ward showed the highest average prices, while the West Ward had the lowest.
- Availability did not show a strong or consistent relationship with rental price.
